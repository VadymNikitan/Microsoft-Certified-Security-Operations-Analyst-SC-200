# SOC-Investigation-Casebook
A growing collection of Microsoft Sentinel and Microsoft Defender XDR investigation case studies based on real-world attack scenarios, Microsoft Threat Intelligence research, and the MITRE ATT&CK framework.

Objectives
Practice Security Operations Center (SOC) investigation workflows
Improve Microsoft Sentinel investigation skills
Develop practical KQL hunting and investigation queries
Map attack scenarios to the MITRE ATT&CK framework
Document investigation methodology and lessons learned
Continuously improve Microsoft Security knowledge through hands-on case studies

Technologies
Microsoft Sentinel
Microsoft Defender XDR
Microsoft Entra ID
Microsoft Defender for Endpoint
Kusto Query Language (KQL)
MITRE ATT&CK

About This Project

This repository documents my journey of developing practical Security Operations Center (SOC) investigation skills using Microsoft security technologies.

Each case study is based on real-world attack scenarios, Microsoft Threat Intelligence reports, or MITRE ATT&CK techniques and includes:

Investigation workflow
Detection logic
Sample KQL queries
Response recommendations
False positives
Lessons learned

Note: The KQL queries provided in this repository are intended as learning examples and investigation starting points. They should be validated and adapted for specific production environments.
